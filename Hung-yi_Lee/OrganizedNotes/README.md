# 深度学习与生成式模型笔记总览

本目录整理现有 Markdown 笔记，按知识体系重编排，便于快速定位与查阅。各文件均为提炼版，但保留公式、推导思路与要点。

- `01_foundations.md`：机器学习基础（回归/分类、梯度下降三种变体、逻辑回归与 GLM、优化器推导）。
- `02_optimization_regularization.md`：激活函数对比、自适应学习率、Momentum/Adam、正则化、Early Stopping、Dropout 等。
- `03_architectures.md`：CNN、RNN/LSTM/GRU、Seq2Seq（AR/NAR）、Attention/Transformer 细节及 Beam Search。
- `04_nlp_models.md`：自监督语言建模（GPT 系列）、BERT 预训练任务与多语对齐、上下文学习与涌现能力、数据工程要点。
- `05_generative_models.md`：Autoencoder/VAE/VQ-VAE、Flow、GAN、Diffusion、图文生成范式、潜变量与采样逻辑。
- `06_semi_supervised.md`：生成式半监督 EM、自训练/熵正则/S3VM、图方法与流形假设，现代 SSL 摘要。
- `07_history_scaling.md`：深度学习发展脉络、经典模型里程碑、Scaling Law、MoE 与大模型现象。

> 所有小节均标注关键公式与适用场景，便于后续扩写与复习。***
